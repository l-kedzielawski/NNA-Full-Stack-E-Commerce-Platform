## Capture Checklist

1. Use the same browser width for all storefront screenshots.
2. Hide any real personal data, emails, phone numbers, or order information.
3. Use demo-safe coupon codes and payment amounts.
4. Prefer clean production-like content over localhost-heavy screenshots.
5. Capture both light and dark homepage states.
6. Include at least one Polish localized route in the browser URL bar.
7. For admin screens, crop tightly so the feature is obvious.
